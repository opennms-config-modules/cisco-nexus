= Clavister

OpenNMS configuration module for Clavister device series.

== Installation

.Download and install from GitHub
[source, bash]
----
mkdir ${HOME}/opennms-config-workspace
cd ${HOME}/opennms-config-workspace
git clone https://github.com/opennms-config-modules/clavister.git
----

=== Data Collection

.Install datacollection configuration
[source, bash]
----
cp ${HOME}/opennms-config-workspace/clavister/datacollection/*.xml ${OPENNMS_HOME}/etc/datacollection
cp ${HOME}/opennms-config-workspace/clavister/resource-types.d/clavister-datacollection.xml ${OPENNMS_HOME}/etc/resource-types.d
----

.Include in datacollection-config.xml
[source, xml]
----
<include-collection dataCollectionGroup="Cisco-Nexus"/>
----

=== Graph Report Definitions

.Install graph report definitions
[source, bash]
----
cp ${HOME}/opennms-config-workspace/clavister/graphs/*.properties ${OPENNMS_HOME}/etc/snmp-graph.properties.d
----
